The tilt metric is the slope of the log rate against wealth rank (quintile midpoints, 10th to 90th percentile) — the same log-odds-style scale OG-Core’s income-group demographics consume, so a value from this library can parameterize the model directly. A slope of −0.79 means the poorest decile’s rate is roughly e0.79×0.8 ≈ 1.9× the richest decile’s.

How a country would borrow from this library

  • Own data first: if the country has a DHS with wealth-quintile tables (78 do), use its own tilt — this library then only brackets it.
  • No own data: borrow the regional median as the central value and treat the regional IQR as the sensitivity band — both improve on the current default of assuming no gradient at all (a tilt of exactly zero, which the library shows is wrong essentially everywhere).
  • Scope: this library covers fertility and under-5 mortality. Adult mortality by wealth cannot be measured by DHS-type surveys (the dead are not interviewed and sibling reports carry no wealth data); its evidence base is panel and registry studies, assembled separately (for South Africa: the NIDS panel estimate of 5% lower death odds per household asset).

Method and sources

  • Data: DHS Program indicator API (api.dhsprogram.com), indicators FE_FRTR_W_TFR (TFR) and CM_ECMR_C_U5M (U5MR), characteristic “Wealth quintile” — 1,560 + 1,451 quintile observations; 601 surveys with complete quintets. Free, public, no registration; fully replicable.
  • Wealth quintiles are the DHS asset index — the standard LMIC proxy for household economic rank (survey income is unreliable; assets rank households well, and the literature finds wealth a stronger mortality predictor than income or education).
  • Tilt = OLS slope of ln(rate) on quintile percentile midpoints (0.10…0.90). “Latest” = most recent survey per country. Regional grouping per the DHS Program.
  • Caveats: quintiles are household-based cross-sections, not lifetime individual rank; the top 1% is not resolved by any quintile design; U5MR quintile cells can be noisy in low-mortality countries (South Africa 2016 visibly so).
